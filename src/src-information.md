# Src 디렉토리 구조 및 기능 설명

## 1. 디렉토리 구조
```
src/
├── app.js              # 애플리케이션 진입점
├── config/            # 설정 파일 디렉토리
├── controllers/       # 컨트롤러 디렉토리
├── core/             # 핵심 기능 디렉토리
├── models/           # 데이터 모델 디렉토리
├── services/         # 비즈니스 로직 서비스 디렉토리
├── tests/            # 테스트 코드 디렉토리
├── utils/            # 유틸리티 함수 디렉토리
└── src-information.md # 현재 문서
```

## 2. 핵심 기능

### 2.1 서버 구성
- Node.js 기반 HTTP 서버
- 정적 파일 서빙 시스템
- MVC 아키텍처 패턴 적용

### 2.2 정적 파일 처리
- 다양한 MIME 타입 지원
- 캐시 정책 구현
- 에러 처리 및 로깅

### 2.3 설정 관리
- 서버 설정
- 환경 변수 관리
- MIME 타입 매핑
